# Hospitality Analytics Dashboard

## Project Overview
This project analyzes hospitality business data to monitor revenue performance, occupancy trends, booking patterns, and customer behavior using interactive dashboards and visual analytics.

The dashboard helps identify business growth opportunities and improve operational decision-making.

---

# Objectives
- Analyze hotel occupancy and booking trends
- Track revenue performance across properties
- Monitor customer behavior and seasonal demand
- Build interactive dashboards for business insights
- Identify opportunities to improve profitability

---

# Tools & Technologies Used
- Power BI / Tableau
- Excel
- SQL
- Python (Optional)

---

# KPIs Analyzed
- Revenue
- Occupancy Rate
- Average Daily Rate (ADR)
- RevPAR
- Booking Trends
- Customer Ratings

---

# Dashboard Features
- Interactive Filters
- Revenue Trend Analysis
- Occupancy Tracking
- Booking Channel Analysis
- Customer Segmentation
- KPI Cards & Charts

---

# Key Insights
- Identified peak booking periods and seasonal trends
- Analyzed occupancy performance across hotel categories
- Compared revenue generation between locations
- Identified customer booking behavior patterns

---

# Business Impact
This dashboard can help hospitality businesses:
- Improve occupancy rates
- Increase revenue performance
- Optimize pricing strategies
- Understand customer preferences
- Improve operational efficiency
